I bought Asus Zenbook UX310UA after carefully considering my needs for setting up a web development environment. I'm planning to learn more about OpenStack, for which I needed a powerful system to spin off instances. MacBooks were pricey for the configuration which I needed. I even considered Dell XPS Developer Edition and System76. Although they are great laptops, Asus offered me a better system for less price.


I've been a long time Ubuntu user. Recently switched to Arch universe, thanks to my good friend Prajakt Shastry. Who has been a pure Arch user for a long time. I looked at various Arch based distros and Antergos seemed quite interesting. It's sleek and easy to install. Most of the things worked right out of the box. I had issues with trackpad but kernel update 4.8.7 has pretty much neutralized it. There are still some issues with palm detection while typing, but they don't affect me intensely. Hotkeys also work right out the box, except for brightness. AUR has been a boon to Arch novice like me. I've found it easy to install packages, precisely because documentation is excellent. I'll work on installing pure Arch and setting up my system from sratch in the near future.


I use Antergos as my primary OS on Asus Zenbook UX310UA. I use it mainly for web development - NodeJS, AngularJS, Redis etc. I also use it to control AWS instances using configured AWS CLI and Vagrant. I ran into some issues while setting up VirtualBox, but they were sorted out once I made sure I had all the required dependencies installed. I do have some issues in configuring my Canon printer. But I'll work on it.
